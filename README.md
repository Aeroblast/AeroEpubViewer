# AeroEpubViewer
An Epub Viewer on Windows. Using C#, CEFSharp.

## 功能
+ 通过第一个命令行参数读入EPUB文件，可以将AeroEpubViewer.exe设置为打开方式。
+ 支持page-progress-direction值rtl。
+ 支持EPUP2.0目录(ncx)(看书的时候点左上角打开目录)
+ 浏览方式为无缝滚动浏览
+ 内存占用大概50~60M。发布档约50M。解压后的二进制占用约200M磁盘空间。
